# Story 7.5 — UI palavras-chave porteiro + parser alfanumérico + nav tab

**Epic:** E7 — Palavra-chave de entrega via WhatsApp
**Status:** Done (retroativa)
**Owner:** @dev (Dex)
**Tipo:** Feature
**Deployed:** 2026-05-21
**Depende de:** epic-7-palavra-chave

---

## User Story

**Como** porteiro,
**eu quero** ver em tempo real as palavras-chave que moradores enviam via WhatsApp e marcá-las como usadas,
**para que** eu saiba quais pacotes estão sendo esperados e registre quando a palavra-chave foi utilizada.

---

## Contexto

O Epic 7 entregou o pipeline backend (parser + worker + persistência). Esta story completa o ciclo adicionando:
1. Parser expandido (aceita palavras alfanuméricas, não só dígitos)
2. API REST para listar/marcar palavras-chave
3. UI do porteiro com polling 10s
4. Tab "Chaves" na bottom nav

---

## Acceptance Criteria

### AC1 — Parser alfanumérico
- [x] `parse-palavra-chave.ts` reescrito para aceitar:
  - Prefixos: `código FACA`, `palavra chave ABC123`, `ML 123456`
  - Mensagens curtas (≤40 chars) como keyword direta
  - Filtro de saudações comuns (oi, olá, bom dia, etc.)
- [x] Backward-compatible com padrões numéricos existentes

### AC2 — API palavras-chave
- [x] `GET /api/palavras-chave` — lista pendentes com `?q=` busca
- [x] `PATCH /api/palavras-chave/[id]` — marca como `consumido`
- [x] Ambos protegidos por `requirePorteiro()`
- [x] Usa `withTenantContext` (RLS)

### AC3 — UI porteiro tempo real
- [x] `PalavrasChaveClient.tsx` com polling 10s
- [x] Busca por apto, morador ou código
- [x] Recentes (<5min) destacadas em verde com indicador pulsante
- [x] Botão "Usado" remove da lista instantaneamente
- [x] Seções "Recentes" e "Anteriores"
- [x] Última atualização visível

### AC4 — DB helper marcar usada
- [x] `marcarPalavraChaveUsada()` em `palavra-chave.ts`
- [x] Seta status=consumido + consumido_em=now
- [x] Idempotente (retorna already_used se já consumido)

### AC5 — Tab bottom nav
- [x] `BottomNavBar.tsx` com 4ª tab "Chaves" (ícone Key)
- [x] href: `/portaria/palavras-chave`
- [x] Layout: Pendentes | Chaves | FAB Chegada | Retirada

### AC6 — Performance
- [x] Removido `force-dynamic` de páginas client-only
- [x] `loading.tsx` com spinner em chegada, retirada, palavras-chave
- [x] Banner feedback `?msg=enviado_administracao` em CapturaPageClient
