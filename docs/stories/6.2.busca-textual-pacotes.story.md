## User Story

**Como** admin, **eu quero** buscar pacotes por nome de morador, número da unidade ou código de rastreio (FR-062), **para que** eu encontre rápido um pacote específico.

---

## Acceptance Criteria

### AC1 — Input de busca em `<PacotesAdminList>`
- [x] Input `<Search>` no topo da tabela
- [x] Param `q` na query string (debounce 300ms client-side)
- [x] Limpar X visível quando há valor

### AC2 — Lookup server-side
- [x] `listPacotesAdmin` aceita `q?: string`
- [x] Quando preenchido, filtra (OR):
  - `nome_destinatario_etiqueta` ILIKE %q%
  - `unidade.identificador` ILIKE %q%
  - `unidade.bloco` ILIKE %q%
  - `codigo_rastreio` ILIKE %q%
- [x] Mínimo 2 chars (senão ignora)

### AC3 — Tests
- [x] Adicionar 2 casos no test integration: busca por nome + busca por unidade

- [x] Lint/typecheck/test/build OK
