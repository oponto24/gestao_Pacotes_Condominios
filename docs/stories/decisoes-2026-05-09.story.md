# Stories combinadas — Decisões produto 2026-05-09

**Status:** Done

## 3 features entregues

### 1. Cancelar pacote (admin_master only)
- `src/lib/db/pacote-cancelar.ts`: helper validando role, motivo obrigatório, status terminal
- `POST /api/pacotes/{id}/cancelar`: endpoint requireAdminMaster
- `CancelarPacoteButton.tsx`: prompt motivo + tratamento 403 amigável
- Integrado no `PacoteDetalheView` quando status != cancelado/retirado

### 2. Lembrete 24h pacote pendente
- Schema: `pacote.ultimo_lembrete_em`, `proximo_lembrete_em`, `lembretes_pausados`, `lembretes_pausados_em`, `lembretes_pausados_motivo` + index parcial
- Migration: ALTER TABLE com defaults seguros
- `enviar-lembretes.ts` job: cron-driven (executar a cada 1h), busca pacotes elegíveis, dispara template QR, agenda +24h
- `send-whatsapp.ts`: ao primeiro envio agenda primeiro lembrete pra +24h
- `process-whatsapp-webhook.ts`: ao receber inbound de morador, pausa lembretes de pacotes pendentes do morador automaticamente
- `POST /api/pacotes/{id}/reagendar-lembrete`: admin_master reagenda data/hora exata após pausa
- **Nota:** scheduler externo (BullMQ repeatable job) precisa ser configurado no worker pra disparar `enviarLembretes` a cada 1h. Configuração inicial: `defaultQueue.add('enviarLembretes', {}, { repeat: { pattern: '0 * * * *' } })` (a fazer no boot do worker)

### 3. Bipe entrega final com aviso (rota administração)
- `RetiradaConfirmarClient`: banner warning quando `pacote.status === 'em_administracao'` informando que pacote estava com admin

## Decisões registradas

- **Cancelamento:** apenas `admin_master` (síndico/admin geral). Não pode cancelar pacote já retirado.
- **Lembrete:** envia template `pacote_chegou` (mesmo do primeiro envio) com novo QR. A cada 24h sem retirada.
- **Pausa automática:** qualquer mensagem inbound do morador pausa todos os pacotes pendentes dele.
- **Reagenda:** admin master escolhe data/hora exata via endpoint (UI pode vir em story futura).
- **Bipe entrega:** banner warning é cosmético — fluxo continua aceitando qualquer role operacional.
