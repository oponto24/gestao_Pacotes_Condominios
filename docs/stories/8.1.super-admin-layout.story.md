## User Story

**Como** super-admin Ponto24 (você + sócios),
**eu quero** uma navegação dedicada em `/super-admin/*` com layout próprio (separado de `/admin/*`),
**para que** fique claro o contexto de operação SaaS vs operação tenant (FR-071).
