# Story 11.4 — Busca global com Cmd+K

**Epic:** E11 — UX Admin Refinado
**Status:** Done
**Owner:** @dev (Dex)
**Estimativa:** 0.5 dia
**Branch:** `main`
**Depende de:** 11.1

---

## User Story

**Como** admin do condominio,
**eu quero** buscar rapidamente moradores, unidades e pacotes com um atalho de teclado,
**para que** eu nao precise navegar entre multiplas paginas pra encontrar informacoes durante o atendimento.

---

## Acceptance Criteria

### AC1 — API GET `/api/admin/search`
- [x] Query param `?q=<termo>` (min 2 chars)
- [x] Busca ILIKE em: morador.nome, morador.telefone, unidade.identificador, pacote.codigo_rastreio
- [x] Max 10 resultados agrupados por tipo
- [x] Tenant-scoped (condominio atual)
- [x] Guard: `requireAdmin()`

### AC2 — Componente `GlobalSearch`
- [x] Botao no header admin com texto "Buscar... ⌘K"
- [x] Cmd+K / Ctrl+K abre dialog de busca
- [x] Input com debounce 300ms
- [x] Resultados com icone por tipo + titulo + subtitulo
- [x] Click resultado → navega + fecha dialog
- [x] Escape fecha
- [x] Loading state + empty state

### AC3 — Integracao no layout admin
- [x] `<GlobalSearch />` visivel no header de `/admin`
- [x] Mobile: icone de busca (sem hint de teclado)
- [x] Desktop: mostra "Buscar... ⌘K"

### AC4 — Quality gates
- [x] Lint, typecheck, test, build OK
