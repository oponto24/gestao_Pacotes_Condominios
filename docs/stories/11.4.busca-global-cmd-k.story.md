# Story 11.4 — Busca global com Cmd+K

**Epic:** E11 — UX Admin Refinado
**Status:** InProgress
**Owner:** @dev (Dex)
**Estimativa:** 0.5 dia
**Branch:** `main`
**Depende de:** 11.1

---

## User Story

**Como** admin do condominio,
**eu quero** buscar rapidamente moradores, unidades e pacotes com um atalho de teclado,
**para que** eu nao precise navegar entre multiplas paginas pra encontrar informacoes durante o atendimento.

---

## Acceptance Criteria

### AC1 — API GET `/api/admin/search`
- [ ] Query param `?q=<termo>` (min 2 chars)
- [ ] Busca ILIKE em: morador.nome, morador.telefone, unidade.identificador, pacote.codigo_rastreio
- [ ] Max 10 resultados agrupados por tipo
- [ ] Tenant-scoped (condominio atual)
- [ ] Guard: `requireAdmin()`

### AC2 — Componente `GlobalSearch`
- [ ] Botao no header admin com texto "Buscar... ⌘K"
- [ ] Cmd+K / Ctrl+K abre dialog de busca
- [ ] Input com debounce 300ms
- [ ] Resultados com icone por tipo + titulo + subtitulo
- [ ] Click resultado → navega + fecha dialog
- [ ] Escape fecha
- [ ] Loading state + empty state

### AC3 — Integracao no layout admin
- [ ] `<GlobalSearch />` visivel no header de `/admin`
- [ ] Mobile: icone de busca (sem hint de teclado)
- [ ] Desktop: mostra "Buscar... ⌘K"

### AC4 — Quality gates
- [ ] Lint, typecheck, test, build OK
