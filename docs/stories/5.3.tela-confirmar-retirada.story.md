## User Story

**Como** porteiro,
**eu quero** ver os dados do pacote que o morador veio buscar e confirmar quem está retirando (próprio destinatário ou terceiro com nome),
**para que** o registro de entrega seja auditado corretamente (FR-042/043/044).

---

## Acceptance Criteria

### AC1 — Página `/retirada/confirmar/[token]`
- [x] Criar `src/app/(portaria)/retirada/confirmar/[token]/page.tsx` (server)
- [x] Carrega pacote via `loadPacoteByQrToken` (5.2)
- [x] Se erro/inválido: redireciona pra `/retirada` com toast (ou 404 friendly)
- [x] Renderiza `<RetiradaConfirmarClient>` com dados

### AC2 — `<RetiradaConfirmarClient>`
- [x] `src/components/portaria/RetiradaConfirmarClient.tsx`
- [x] Mostra:
  - Foto thumbnail
  - Nome destinatário em destaque
  - Unidade (Bloco X · Apto Y)
  - Setor + posição (onde está armazenado — porteiro vai buscar)
  - Remetente (info útil)
- [x] Pergunta destacada: **"O próprio destinatário está retirando?"**
- [x] 2 botões grandes (h-14): **"Sim, é {nome}"** (verde/primary) | **"Não, outra pessoa"** (ghost)

### AC3 — Sheet "outra pessoa"
- [x] Quando clica "Não, outra pessoa" → abre Sheet/Drawer com:
  - Input "Nome de quem está retirando" (obrigatório, max 200 chars)
  - Botão "Confirmar entrega" (faz PATCH 5.4)
  - Botão "Voltar"
- [x] Validação: nome >= 3 chars

### AC4 — Submit
- [x] Envia `PATCH /api/pacotes/{id}/retirar/confirmar` (5.4) com body:
  - `proprio_destinatario: boolean`
  - `retirado_por_terceiro?: string` (se proprio=false)
- [x] Sucesso → redireciona `/retirada/sucesso?pacote_id={id}` (página simples de feedback)
- [x] Erro → toast vermelho

### AC5 — Página de sucesso
- [x] Criar `src/app/(portaria)/retirada/sucesso/page.tsx`:
  - Mensagem grande "Entregue com sucesso!" + ícone CheckCircle2
  - Botão "Próxima retirada" → `/retirada`
  - Botão "Voltar pra chegada" → `/chegada`

### AC6 — Tests
- [x] Mantém AC apenas de comportamento — sem testes de DOM (igual padrão 3.6/3.8)

### AC7 — Quality gates + smoke
- [x] Lint, typecheck, test, build OK
- [x] Smoke S1-S3 manual sob demanda
